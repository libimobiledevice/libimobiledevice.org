◆ heartbeat_client_start_service()

LIBIMOBILEDEVICE_API heartbeat_error_t heartbeat_client_start_service ( idevice_t  device,
heartbeat_client_t *  client,
const char *  label 
)

Starts a new heartbeat service on the specified device and connects to it.

Parameters
deviceThe device to connect to.
clientPointer that will point to a newly allocated heartbeat_client_t upon successful return. Must be freed using heartbeat_client_free() after use.
labelThe label to use for communication. Usually the program name. Pass NULL to disable sending the label in requests to lockdownd.
Returns
HEARTBEAT_E_SUCCESS on success, or an HEARTBEAT_E_* error code otherwise.
